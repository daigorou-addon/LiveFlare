# LiveFlare

Live stream monitor & alert app for Windows.

## Features
- Real-time online status display
- Desktop toast notifications when stream starts
- Twitch follow list import
- 13 language support
- System tray background operation

## Supported Platforms
YouTube / Twitch / Kick / TwitCasting / FC2 Live

## Requirements
Windows 10/11 (64bit)
